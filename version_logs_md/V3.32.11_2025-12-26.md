# V3.32.11 版本更新说明

### 新增

- [ 新增 ] 增加接口获取任务执行的有效时长（用于AI提效统计）
- [ 新增 ] 模板信息网关接口返回添加备注信息
- [ 新增 ] get_task_list 接口支持通过时间和创建方式来进行过滤
- [ 新增 ] 接入ai小鲸，支持查询标准运维产品使用知识库
- [ 新增 ] Job部分内置插件支持host_id作为服务器参数
- [ 新增 ] HTTP回调提供批量更新接口

- [ 优化 ] 公共流程可见范围增加复制粘贴按钮
- [ 优化 ] 网关MCP Server对返回的信息进行过滤

- [ 修复 ] 回调请求头信息被隐藏问题修复
- [ 修复 ] 新建任务第二步情况下跳转错误修复
