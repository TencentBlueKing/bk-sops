# API

## 调用指引
所有的 API 在正式环境调用都要通过蓝鲸API网关调用，请按照你部署的蓝鲸PaaS平台的"开发者中心"->"API网关"->"使用指南"操作。

点击这里查看[API 列表](sops.yaml)。
